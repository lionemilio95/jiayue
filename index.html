    Projects

    Pokemon World

    This is a 3D animation program based on OpenGL and C++. My team had implemented toon shading, illumination effects, physical collision(using bouding spheres). We also applied Diamond Algorithms and L-system to procedurally generate terrian and plants to create the virtual Pokemon World. The wave effects were realized by using bezier curves in motion.

    Android Robot Squad

    This project is implemented by OpenGL and C++. I created a squad of moving Android robots (over 400) in one scene and add texture on them. In order to reduce the burden of GPU&CPU, I made a culling function that it would only render the robots inside the camera. I also created bounding sphere for them to detect if the robots are colliding each other.

    Bezier Roller Coaster

    This project is implemented by OpenGL and C++. I created a track and a sphere to simulate the movement of a Roller Coaster. The track has followed the pattern of Bezier path (as shown in yellow handler) and the roller coaster moves fast when it goes down and move slow when it moves up along with the track. There is also a sky box which contains the track and the roller coaster. The sphere roller coaster has a mirror texture which reflects the sky box.

    Prediction with limited trainning data

    This is a kaggle playground project with Machine Learning. In this project we were given extremly limited data to train the learning model. We tried using XGBoosting, Random Forest Model, SVC model, SGD model, Logistic Regression and Neural Network and tuning these models to predict the test result. However, none of them works well. Then we tried the LB_Score, trying to improve the accuracy and it finally works.
